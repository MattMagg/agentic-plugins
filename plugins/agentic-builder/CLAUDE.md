# CLAUDE.md - Agentic Builder Plugin

This file provides guidance to Claude Code when working with the agentic-builder plugin.

## Plugin Overview

Framework-agnostic agentic system builder with RAG-grounded knowledge. Supports:
- **Google ADK** (Agent Development Kit)
- **OpenAI Agents SDK**
- **LangChain/LangGraph**
- **Anthropic Agents SDK**
- **CrewAI**

## Commands

- `/agentic` - Main entry point. Auto-detects framework, adapts to task type (build, debug, plan, etc.)
- `/agentic --debug` - Enter debugging mode
- `/agentic --spec` - Force spec/requirements gathering mode
- `/agentic --plan` - Force planning mode
- `/agentic --framework adk` - Specify framework explicitly

## MCP Tools

The plugin provides RAG query tools via MCP:
- `mcp__agentic-rag__query_docs` - Search documentation
- `mcp__agentic-rag__query_code` - Search code examples
- `mcp__agentic-rag__search_patterns` - Find implementation patterns
- `mcp__agentic-rag__list_frameworks` - List available frameworks

## Skills

Framework skills (idioms, gotchas, quick patterns):
- `@adk` - Google ADK patterns
- `@openai-agents` - OpenAI Agents SDK patterns
- `@langchain` - LangChain patterns
- `@langgraph` - LangGraph patterns
- `@anthropic-agents` - Anthropic Agents patterns
- `@crewai` - CrewAI patterns

Task skills:
- `@debugging` - Cross-framework debugging
- `@deployment` - Production deployment
- `@orchestration` - Multi-agent patterns

## State Management

Session state persists in `.claude/agentic-builder.local.md`:
- Active framework(s)
- Current phase (spec/plan/build/debug)
- Last completed step
- Key decisions (for post-compact restoration)
